Architecture

The pipeline fills missing metadata fields for archaeological artifacts from the OpenContext database using a retrieval-augmented multimodal generation approach.

Encoding: Each v4 artifact is encoded as a 1408-dim vector — DINOv2 ViT-L/14 image embedding (1024-dim) concatenated with all-MiniLM-L6-v2 text embedding (384-dim) from available metadata fields.

Index: FAISS flat index (IndexFlatIP) built on the 85% train split of v4 (19,215 artifacts). The remaining 15% (3,392 artifacts) are held out as the eval set — never seen by the index.

Retrieval: Top-N most similar artifacts retrieved, filtered to only those with the target field populated.

Generation: GPT-4o receives the artifact image + available fields + retrieved neighbors as structured JSON context, plus a constrained vocabulary derived from train split distributions.
